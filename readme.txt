== Description ==

A simple wordpress image slider plugin. A responsive jQuery slideshow with beautiful panning effects plugin. It's very easy to use with a simple shortcode.

= Demo =

Go to [Plugin Demo Site](http://sumonhasan.com/plugins/b26-slider/)

= Usage =
Follow the instructions below.

1. Go B26 Slider > Add New.  

1. Give a title and upload image from select your image for B26 Slider.

1. Create a category and select the category. And publish the post.

1. Add new image as much as you want to publish.

1. Use this shortcode in your post or page or widget.

<code>[b26slider]</code>

1. To use this shortcode in your template, write 

<code><?php echo do_shortcode('[b26slider]'); ?></code>

= More Usage Info =

Use this code to get to publish your B26 Slider from a particulat category and slider image count.

<code>[b26slider category="Your_Category_Name" count="4"]</code>



== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

B26 Slider Menu will appear in admin menu. Add slider image and create a category. To publish the slider use shortcode in a post or a page or a template.
